# E-Commerce Customer Analytics — Behavioral & Transactional Insights

## Project Overview
This project analyzes over 500,000 customer transactions to understand purchasing behavior, spending patterns, churn tendencies, and overall business trends. It includes structured data cleaning, transformation, visual exploration, and extraction of actionable insights to support data-driven decision-making for marketing, sales, and customer retention strategies.

---

## Dataset Summary
- Total Records: 500,000  
- Includes both numerical and categorical features  
- Contains demographic and transaction attributes

### Core Features
| Column | Description |
|--------|-------------|
| Product_Category | Type of product purchased |
| Product_Price | Price per product |
| Quantity | Units purchased |
| Total_Purchase_Amount | Final amount paid |
| Payment_Method | Mode of payment used |
| Customer_Age | Age of the customer |
| Gender | Male/Female |
| Returns | Whether the product was returned |
| Churn | Customer retention indicator |

---

## Data Preparation & Processing
- Merged and combined datasets into a single dataframe  
- Standardized column names  
- Converted date fields to datetime  
- Ensured numeric types for relevant attributes  
- Verified dataset consistency and completeness  
- Confirmed absence of duplicate records  
- Cleaned and prepared data for analysis

---

## Exploratory Data Analysis (EDA)
Analytical techniques applied include:
- Statistical distribution analysis  
- Frequency analysis for categorical variables  
- Correlation analysis across numerical features  
- Relationship exploration between price and purchase amount  
- Purchase trends by age, gender, and product type  
- Visualization-driven insights using histograms, boxplots, bar charts, and heatmaps

---

## Key Observations
- Clothing and Books are the most frequently purchased categories  
- Majority of customers fall between ages 25–55  
- Credit Card and PayPal are the most common payment methods  
- Spending behavior is nearly identical across genders  
- Higher spending is associated with lower churn  
- Returns appear relatively rare in the dataset  
- Dataset is well-distributed and suitable for modeling
